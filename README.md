# Simple RAG Chat PDF

This is a simple RAG (Retrieval-Augmented Generation) application built with Streamlit, LangChain, and FAISS.

## Features
- Upload PDF files
- Chat with the content of the PDF
- Local vector storage using FAISS
- Conversational memory

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
